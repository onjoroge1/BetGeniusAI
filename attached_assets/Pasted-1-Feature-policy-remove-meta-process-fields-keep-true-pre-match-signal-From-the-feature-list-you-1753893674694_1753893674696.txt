3) Fix Brier + Top‑2 implementations (single source of truth)
Your “Brier 0.535 / 0.956” indicates the unnormalized variant (sum over classes, no divide by K). Use the normalized multiclass Brier:

def brier_multiclass(y_true, proba, labels=('Home','Draw','Away')):
    import numpy as np
    y = np.eye(len(labels))[np.array([labels.index(t) for t in y_true])]
    # normalized: average across classes (K) and samples (N)
    return ((proba - y)**2).mean()
For uniform 33/33/33, normalized Brier ≈ 0.222.

Gate remains Brier ≤ 0.205.

def top2_acc(y_true, proba, labels=('Home','Draw','Away')):
    import numpy as np
    top2 = np.argsort(-proba, axis=1)[:, :2]
    idx  = np.array([labels.index(t) for t in y_true])
    return ((top2[:,0] == idx) | (top2[:,1] == idx)).mean()

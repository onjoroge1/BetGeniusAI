## What we should do

### 1. Create a “results + closing odds join table”

Something like:

```sql
CREATE TABLE odds_snapshot_results AS
SELECT os.match_id,
       os.book_id,
       os.outcome,
       os.implied_prob,
       os.created_at,
       mr.outcome AS actual_result,
       c.ph_close, c.pd_close, c.pa_close
FROM odds_snapshots os
LEFT JOIN match_results mr ON os.match_id = mr.match_id
LEFT JOIN clv_closing_feed c ON os.match_id = c.match_id;
```

This table will be your **single source of truth** for evaluation:

* if `actual_result` is NULL → match not finished yet
* if `ph_close` etc. are NULL → closing odds not captured yet

Then your accuracy test script can just hit this table.

## New tools you can run now

### 1) **Instance-wise book mixing (softmax gating)**

**File:** `book_mixer_trainer.py`
**Download:** [book\_mixer\_trainer.py](sandbox:/mnt/data/book_mixer_trainer.py)

**What it does**

* Learns **per-match weights** $w_b(x)$ for each bookmaker via a **softmax gating** model using your **context features** (e.g., dispersion, n\_books, overround, has\_pinnacle, league/phase).
* Blends books with a **geometric mixture**:

  $$
  p_{\text{mix}}(o)\ \propto\ \prod_b p_b(o)^{w_b(x)}\quad\Longleftrightarrow\quad
  \log p_{\text{mix}}=\sum_b w_b(x)\,\log p_b
  $$
* Optimizes **multiclass NLL**, outputs:

  * **Metrics** vs equal-geom and static weighted baselines (LogLoss, Brier, Top-2).
  * **Per-match learned weights** and mixed consensus `pH_mix/pD_mix/pA_mix`.

**Input CSV format (wide)**

* `y` in {0,1,2} (H/D/A).
* For each book b: `pH_b, pD_b, pA_b` (implied probs at horizon; script normalizes).
* Optional context features: any columns prefixed `ctx_…` (e.g., `ctx_dispersion`, `ctx_n_books`, `ctx_overround`, `ctx_has_pinnacle`, `ctx_league_id_*`, `ctx_season_phase_*`).

**Usage**

```bash
python book_mixer_trainer.py \
  --data /path/to/per_match_books.csv \
  --epochs 400 --lr 0.05 --l2 1e-3 \
  --outdir ./book_mixer_artifacts
```

*Smoke tests pass with synthetic data if `--data` is omitted.*

**Input CSV format (long)**

**Usage**

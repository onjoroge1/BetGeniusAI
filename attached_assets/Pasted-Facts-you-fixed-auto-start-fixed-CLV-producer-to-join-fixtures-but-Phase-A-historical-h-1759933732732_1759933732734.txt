Facts: you fixed auto-start ✅, fixed CLV producer to join **fixtures** ✅, but Phase A (historical) hogs the collector so Phase B (upcoming odds) never gets a turn. Let’s flip the execution model so **fresh odds always win** and historical work is cooperative, time-boxed, and resumable.

Here’s a surgical, production-safe plan with code patterns you can drop in.

# 1) Split jobs and give Phase B hard priority

**Do not** run Phase A and Phase B in one long sequential “collector.” Register them as separate scheduled jobs, and make Phase B higher frequency:

* Phase B: every **30–60s** (strict deadline ≤ 3s runtime)
* Phase A: every **5–10 min** (time-slice ≤ 20–30s per tick)

```py
# scheduler.py (concept)
sched.every(30).seconds.do(run_upcoming_odds_phase_b)   # HIGH PRIORITY
sched.every(10).minutes.do(run_historical_phase_a)      # LOW PRIORITY
```

# 2) Time-box each job (deadlines) so they never overrun

Use a wall-clock budget per run. If you hit the budget, **persist a cursor** and exit; pick up next run where you left off.

```py
# utils/timebox.py
import time
class Timebox:
    def __init__(self, seconds): self.end = time.time() + seconds
    def ok(self): return time.time() < self.end

# models/automated_collector.py
async def run_upcoming_odds_phase_b():
    tb = Timebox(3)  # hard 3s budget
    # fetch fixtures in [now, now+6h], by kickoff ASC
    for fx in await fetch_upcoming_fixtures(limit=100):
        if not tb.ok(): break
        await collect_latest_odds_for_fixture(fx.match_id)  # writes odds_snapshots

async def run_historical_phase_a():
    tb = Timebox(30)  # soft 30s budget
    cursor = await load_cursor("hist_league_cursor")  # {league_id, from_dt}
    for league in round_robin_leagues(start=cursor.league_id):
        if not tb.ok(): break
        cursor = await collect_historical_slice(league, cursor.from_dt, slice_minutes=15)
    await save_cursor("hist_league_cursor", cursor)
```

# 3) True async: don’t block the loop; fan-out with `gather` + small batches

```py
import asyncio

async def run_upcoming_odds_phase_b():
    tb = Timebox(3)
    fixtures = await fetch_upcoming_fixtures(limit=30)
    batch = []
    for fx in fixtures:
        if not tb.ok(): break
        batch.append(collect_latest_odds_for_fixture(fx.match_id))
        if len(batch) == 6:                  # tiny parallelism
            await asyncio.gather(*batch)
            batch.clear()
    if batch:
        await asyncio.gather(*batch)
```

# What broke (root cause)

* **Constraint:** `odds_snapshots.market_margin` is `NOT NULL`.
* **API-Football path:** we were inserting odds rows **without** computing a `market_margin` → first insert fails → the whole transaction gets marked **aborted** → every subsequent insert errors (`current transaction is aborted…`).

The Odds API path already computes a margin; our API-Football path needs the same.

---

# Minimal, surgical fix

## 1) Add overround/margin calculation (once per market)

```python
# utils/api_football_integration.py (or a shared utils module)

from typing import List, Optional

def compute_market_margin(prices: List[Optional[float]]) -> Optional[float]:
    """Return market overround (sum(1/price) - 1). Ignore None / <=0 prices."""
    inv_sum = 0.0
    count = 0
    for p in prices:
        if p and p > 0:
            inv_sum += 1.0 / p
            count += 1
    # Need at least two valid outcomes to call this a market
    return max(inv_sum - 1.0, 0.0) if count >= 2 else None

def implied_prob(price: Optional[float]) -> Optional[float]:
    return (1.0 / price) if price and price > 0 else None
```

## 2) In the API-Football ingestion, group outcomes by (fixture_id, bookmaker, market)

* Compute `market_margin` **once** from all valid selections’ prices.
* Insert each selection with the same `market_margin`.
* If margin is `None` (not enough outcomes), **skip** those rows (or set to `0.0` as a last resort — I recommend skip + log).

```python
# utils/api_football_integration.py (inside your ingestion loop)
# Pseudocode sketch – adapt to your existing variables

    margin = compute_market_margin(prices)

              "selection": s["selection"],            # 'H' / 'D' / 'A'
              "price": price,
              "implied_prob": prob,
              "market_margin": margin,                # <-- FIX
```

## 3) Make inserts resilient to one-row failures (no “aborted transaction” cascade)

If you use psycopg, wrap rows or batches with **SAVEPOINT** to isolate failures:

```python
# using psycopg (v3) style
with conn.cursor() as cur:
    for payload in payloads:
        cur.execute("SAVEPOINT sp_ins")
        try:
            cur.execute(SQL, payload)
        except Exception as e:
            logger.error(f"Row insert failed, rolling back savepoint: {e}")
            cur.execute("ROLLBACK TO SAVEPOINT sp_ins")
        finally:
            cur.execute("RELEASE SAVEPOINT sp_ins")
conn.commit()
```

(If you’re using an ORM or a helper `db` wrapper, emulate the same semantics: per-row try/except + continue.)

```python
# utils/names.py
import unicodedata

def strip_diacritics(s: str) -> str:
    return ''.join(c for c in unicodedata.normalize('NFKD', s) if not unicodedata.combining(c))

def slug_team(s: str) -> str:
    s = strip_diacritics(s or "").lower()
    keep = "abcdefghijklmnopqrstuvwxyz0123456789 "
    s = ''.join(ch if ch in keep else ' ' for ch in s)
    return ' '.join(s.split())

def canonicalize_team(name: str, provider: str, alias_map: dict) -> str:
    k = (provider, slug_team(name))
    return alias_map.get(k, name)
```

## 0) One-screen reality check (define & recompute LogLoss/Brier)

Have the agent re‑compute metrics with one shared reference:

```python
import numpy as np

def logloss_mc(y_true_idx, P, eps=1e-15):
    # y_true_idx: int labels {0,1,2} for H/D/A
    # P: Nx3 predicted probs, rows sum to 1
    p_true = np.clip(P[np.arange(len(P)), y_true_idx], eps, 1-eps)
    return float(-np.log(p_true).mean())

def brier_mc(y_true_idx, P):
    Y = np.eye(3)[y_true_idx]
    return float(((P - Y)**2).mean())  # normalized (avg over classes and samples)
```

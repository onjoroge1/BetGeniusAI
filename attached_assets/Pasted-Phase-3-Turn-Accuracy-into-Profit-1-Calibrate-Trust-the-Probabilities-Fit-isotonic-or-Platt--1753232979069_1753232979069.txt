Phase 3: Turn Accuracy into Profit

1. Calibrate + Trust the Probabilities
	•	Fit isotonic or Platt scaling on a holdout fold; save calibrator.
	•	Produce reliability plots & Brier score each retrain.
	•	Use calibrated probs for EV and staking.

2. Add the Betting Layer (if not done)
	•	Ingest opening (and ideally closing) odds.
	•	Compute implied probs (margin-adjusted).
	•	For each outcome: edge = p_model - p_implied.
	•	Filter bets: choose threshold (e.g., edge ≥ 3–5%) to control volume.
	•	Track: ROI, hit rate, avg EV, CLV (closing line value).

3. Optimize Thresholds

Run a sweep:
	•	Edge cutoffs (1% → 10%)
	•	Min probability floors
	•	Output: curve of ROI vs #bets → pick sweet spot.

4. Monitor in Prod

Daily/weekly job to:
	•	Recompute rolling LogLoss, Top-2, ROI, CLV.
	•	Drift alerts (class dist, feature shifts).
	•	Auto-report league/season breakdowns.

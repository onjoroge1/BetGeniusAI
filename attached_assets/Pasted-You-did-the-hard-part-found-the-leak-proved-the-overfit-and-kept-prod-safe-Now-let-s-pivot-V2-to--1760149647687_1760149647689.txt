## 1) Replace the ensemble with a simple “market-adjustment” model

**Idea:** Don’t predict raw 1×2 probs. Learn **small deltas around the market** in logit space, then cap them.

### Training (use your existing temporal split)

* Inputs (minimal, robust):
  `prob_home, prob_draw, prob_away, overround, book_dispersion, drift_24h_*` (+ any extra non-null features).
* Targets: actual outcome `y ∈ {H,D,A}`.
* Build **market logits**:

  ```
  z_m = [log(ph), log(pd), log(pa)] - logsumexp([log(ph),log(pd),log(pa)])
  ```
* Train a **ridge (L2) multinomial logistic regression** to predict **delta logits**:

  ```
  Δz = W·X + b   (3 outputs)
  ```
* **Clamp** each component: `Δz = clip(Δz, -τ, +τ)` with τ=0.5 (≈ ±15-20% prob swings).
* Final logits: `z = z_m + α·Δz`, with **α=0.5** (blend weight).
* Softmax → `p̂` (then isotonic calibration on validation only, as you already do).

> Why this works: the market remains the strong prior; the model can only make **bounded, explainable** nudges where features disagree with the market.

### Minimal code (training core)

```python
# X: features, y: {0,1,2}, pm=(ph,pd,pa)
import numpy as np
from sklearn.linear_model import LogisticRegression

def market_logits(pm):
    l = np.log(np.clip(pm, 1e-6, 1-1e-6))
    l = l - np.logaddexp.reduce(l, axis=1, keepdims=True)
    return l

def train_delta_logit(X_tr, y_tr, X_val, y_val, pm_tr, pm_val, alpha=0.5, tau=0.5):
    # multinomial ridge
    clf = LogisticRegression(
        penalty='l2', C=0.5, solver='lbfgs', max_iter=2000,
        multi_class='multinomial', random_state=42
    )
    clf.fit(X_tr, y_tr)

    def predict(pm, X):
        zm = market_logits(pm)               # prior logits
        dz = clf.decision_function(X)        # shape (n,3)
        dz = np.clip(dz, -tau, tau)          # hard clamp
        z  = zm + alpha * dz                 # blend
        z -= z.max(axis=1, keepdims=True)    # stabilize
        p  = np.exp(z); p /= p.sum(1, keepdims=True)
        return p

    p_val_raw = predict(pm_val, X_val)

    # fit isotonic calibrators on validation only
    from sklearn.isotonic import IsotonicRegression
    iso = [IsotonicRegression(out_of_bounds='clip') for _ in range(3)]
    for k in range(3):
        iso[k].fit(p_val_raw[:,k], (y_val==k).astype(float))

    def apply_cal(p):
        out = np.column_stack([iso[k].predict(p[:,k]) for k in range(3)])
        out = np.clip(out, 0.02, 0.98)
        out /= out.sum(1, keepdims=True)
        return out

    p_val = apply_cal(p_val_raw)
    return clf, iso, predict, apply_cal
```

# Phase 0 — Baseline Hygiene (do first)

**BUILD CARD 0.1 — Normalize V1 on read + block time-leak**

```
Objective: Make metrics airtight.

Create/Update:
- services/metrics/utils.py:
  - def normalize_triplet(p_home, p_draw, p_away) -> (ph,pd,pa):
      s = max(1e-9, p_home + p_draw + p_away); return (p_home/s, p_draw/s, p_away/s)

- services/metrics/evaluator.py:
  - When loading rows for ANY model_version:
      (p_home, p_draw, p_away) = normalize_triplet(...)
  - Filter eval set: scored_at < kickoff_time (LEFT JOIN fixtures)
  - Add 'filtered_out_reason' counters (post_kickoff, missing_kickoff).

Outputs:
- /metrics/evaluation now reports: total_evaluated, dropped_post_kick, dropped_missing_kick.
- Unit test: tests/metrics/test_normalization.py (V1 sums 1.18 → 1.0).

Acceptance:
- V1/V2 prob sums ∈ [0.999, 1.001] for 100% rows.
- post-kickoff rows excluded in results.
```

**BUILD CARD 0.2 — Add ECE + Reliability Curves**

```
Objective: Calibration visibility.

Create:

  - def reliability_table(y_true, P, n_bins=10) -> List[{bin, avg_max_p, true_freq, count}]

Acceptance:

```

# Phase 1 — Stepwise V2 Core (this sprint)

**BUILD CARD 1.1 — Multiclass Temperature Scaling Module**

```
Objective: Fix over/under-confidence without changing rankings.

Create:
- models/calibration/temp_scaler.py (PyTorch or NumPy)
  - class TempScaler:
      fit(logits, y) -> learns scalar T (per-global)
      predict_proba(logits) -> softmax(logits / T)

Storage:
- Save learned T per "cohort": global + per-league (optional).
  - table: model_calibration (model_version, cohort_key, T, fitted_at)

Wire:
- prediction_service for V2: apply TempScaler at inference if cohort T exists.

Outputs:
- CLI: python tools/fit_temp_scaler.py --model v2 --window 90d --by-league
- Writes T(s) and logs pre/post ECE & log-loss deltas.

Acceptance:
- On last 241 matches (paired set), log-loss ↓, ECE ↓ or stable; hit% unchanged or ↑.
```

**BUILD CARD 1.2 — Blend/Constraint Sweep (α, KL, τ)**

```
Objective: Stop over-dispersion; find better confidence.

Create:
- experiments/v2_blend_sweep.py
  - Grid:
    alpha ∈ {0.5, 0.6, 0.7, 0.8}
    kl_cap ∈ {0.15, 0.20, 0.25}
    delta_tau ∈ {0.75, 1.0}
  - For each config:
    - re-score validation slice (recent 60-90d),
    - apply TempScaler(T_global),
    - compute: log-loss, Brier, hit, ECE, avg_max_p, by-league slices.

Outputs:
- artifacts/sweeps/v2_blend_sweep.json (sorted by log-loss).

Acceptance:
- Best config beats current V2 log-loss (paired); Brier ≤ current; ECE green (≤0.03 global).
- Persist chosen config in CONFIG: config/v2_inference.json.
```

**BUILD CARD 1.3 — Draw-Aware Prior (optional but small win)**

```
Objective: Place more realistic mass on draws when teams are even.

Create:
- models/postprocess/draw_prior.py
  - if entropy(P_market) ≥ H_thr and |ph - pa| ≤ Δ_thr:
      p_draw ← p_draw + λ * (target_draw_rate_league - p_draw)
  - renormalize.

Params (configurable per league): H_thr, Δ_thr, λ ∈ [0.1, 0.3].

Wire:
- V2 postprocess pipeline before temperature scaling.

Acceptance:
- Slight ↑ in draw calibration (bin-level), overall log-loss ≤ previous step.
```

**BUILD CARD 1.4 — Paired Significance Tests**

```
Objective: Prove gains aren’t vibes.

Create:
- tools/paired_tests.py
  - McNemar(V1_vs_V2_hits), bootstrap deltas for log-loss & Brier with 10k resamples.
- /metrics/evaluation?compare=v1,v2 adds: { "mcnemar_p":..., "delta_logloss_CI": [l,u] }

Acceptance:
- Script runs locally; endpoint surfaces results for the last evaluated window.
```

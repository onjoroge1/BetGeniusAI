## EXPLORATION RESULTS

### ✅ API-Football Capabilities

1. **Bookmakers Endpoint**: WORKING
   - 31 bookmakers available
   - Clean IDs and names (Bet365=8, Pinnacle=4, William Hill=7, etc.)
   - Mix of European, UK, US, and Asian bookmakers

2. **Odds Endpoint**: PARTIALLY WORKING
   - Historical odds available for SOME matches (20% success rate in sample)
   - Successfully retrieved odds for: Atalanta vs Club Brugge (UEFA CL)
   - 12 bookmakers returned with Home/Draw/Away decimal odds
   - Data format: Compatible with our odds_snapshots table

3. **Data Format**:
   ```json
   {
     "bookmaker": {"id": 1, "name": "10Bet"},
     "bet": {
       "name": "Match Winner",
       "values": [
         {"value": "Home", "odd": 1.83},
         {"value": "Draw", "odd": 3.95},
         {"value": "Away", "odd": 3.85}
       ]
     }
   }
   ```

### 📊 Bookmaker Coverage Analysis

**API-Football** (31 bookmakers):
- Bet365, Pinnacle, William Hill, 1xBet, Unibet, Ladbrokes, Betway
- Marathonbet, Betfair, Bwin, Dafabet, BetFred, 188Bet
- Interwetten, 5Dimes, Intertops, Bovada, Betcris, 888Sport
- Tipico, Sportingbet, Expekt, Betsson, NordicBet, ComeOn
- Netbet, Betano, Fonbet, Superbet
- + more

**The Odds API** (current system):
- Uses numeric book_ids (0-1000+) without name lookup
- 907 unique IDs in odds_snapshots table
- No existing mapping to bookmaker names
- Examples: 777, 628, 937, 860, 165, 243

**Key Finding**: NO OVERLAP
- Current system uses arbitrary numeric IDs
- API-Football uses clean IDs with names
- No existing crosswalk table to map between them
